Stop losing tabs.

Pin Context turns pinned files into a fast, keyboard-first navigation flow with dedicated contexts, timeline history, and scalable behavior for large projects.

Install in VS Code View Source
TypeScript VS Code >= 1.85 MIT License

Core capabilities

Fast pin operations

Pin, unpin, toggle current editor, bulk pin all tabs, and use glob-based pinning.

Workflow contexts

Create task-focused contexts and switch between them without losing your working set.

Quick access UX

Use quick picks, command palette actions, and fuzzy search for pinned files.

Context timeline

Jump back to recently used contexts grouped by time buckets.

Dedicated sidebar

Work with active context, all contexts, recent timeline, and pinned files in one place.

Scales with projects

Batching and configurable limits keep interaction responsive in large workspaces.

Popular commands

Pin Current Editor

Pin the active tab instantly.

Ctrl + Cmd + K

Unpin Current Editor

Remove pin from active tab.

Ctrl + Cmd + U

Toggle Pin Current Editor

Switch pin state in one shortcut.

Alt + Cmd + K

Typical workflow

  1. Pin files related to your current task.
  2. Create a context and give it a clear name.
  3. Switch contexts as you move between tasks.
  4. Use timeline to jump back to recent work.
  5. Quick-open pinned files with fuzzy search.
